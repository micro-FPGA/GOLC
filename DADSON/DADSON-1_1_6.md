DADSON-1
By Antti Lukats, 6 AG

DADSON-1 was the codename for DAD's designed computer for his SON. The son also helped a little bit with the design. 
The project schematic was completed, from PCB work was maybe 25% done; the board was never actually manufactured.

https://github.com/AnttiLukats/Antti-Bible/tree/main/extras/FPGA/DadSon

The board was targeted to be low cost, with an easy-to-use extension connector and some peripherals:
* HDMI
* 4x USB Host
* 1G Ethernet
* micro-SD Card slot

Other features would be
* CPU Dual core ARM A9
* DDR3 16-bit DDR memory
* Linux capable system
