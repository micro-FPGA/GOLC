🏆 Grand Old Lady’s Contest

A celebration of creativity, tradition, and innovation.

About the Contest

The Grand Old Lady’s Contest (GOLC) invites creators, thinkers, and innovators to reinterpret legacy through modern storytelling, design, or code.

How to Participate

  1. Fork the repository on GitHub.
  2. Add your submission under the submissions/ folder.
  3. Submit a pull request before the deadline.

Contact

For inquiries, open an issue on GitHub or contact the organizers directly.
